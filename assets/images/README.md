# Favicon Images

This directory should contain the following favicon files:

- `favicon.ico` - Main favicon file (16x16, 32x32, 48x48)
- `favicon-16x16.png` - 16x16 PNG favicon
- `favicon-32x32.png` - 32x32 PNG favicon
- `apple-touch-icon.png` - 180x180 PNG for Apple devices

## How to Generate Favicons

You can generate these favicon files using online tools like:
- https://favicon.io/
- https://realfavicongenerator.net/
- https://www.favicon-generator.org/

Use a book or comic-related icon/logo for the ComicVerse brand.
